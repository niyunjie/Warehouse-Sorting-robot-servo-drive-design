# 机器人伺服驱动控制系统综合设计

本仓库整理了综合设计作业相关的报告、Simulink 模型和仿真结果。

## 内容

- `1320231081-倪蕴杰-综合设计作业.pdf`：最终课程设计报告。
- `机器人伺服驱动控制系统设计_A.docx`：课程设计文档源文件。
- `1320231081_math_model.slx`：数学模型仿真文件。
- `1320231081_physical_model.slx`：物理模型仿真文件。
- `figure/`：报告中使用的图像和绘图源文件。
- `仿真图/`：数学模型和物理模型的仿真结果图。
- `工况.txt`：仿真工况说明。

## 说明

项目文件主要用于课程作业归档和复现实验结果。打开 `.slx` 文件需要 MATLAB/Simulink 环境。
